Select and label training data...

This side can be used to train a neural network to select the images and sort out images useful for the study. This is useful in situations where we have many images and it is tedious to select them by hand. It is often a good idea, initially, to avoid selecting questionable images and start with typical examples. Depending on the outcome of the first training round you can then select images that are closer to the border between the concepts. This process can be described as "curriculum learning" in a human context, initially, you start with establishing the expected concepts clearly and then, in later stages, the more fine-grained the learning experience becomes. The "other" concept can be used, as a sort of trashcan, to collect all images that do not belong to any of the named concepts regardless of the inhomogeneity you, as a human, find the concept "other" entails.
